# modelClassroom

A Python library for LLM operations and utilities.

## Installation

### From source

```bash
git clone https://github.com/yourusername/modelclassroom.git
cd modelClassroom
pip install -e .
```

### From PyPI (when published)

```bash
pip install modelclassroom
```

## Development

### Setup development environment

```bash
pip install -e ".[dev]"
```

### Run tests

```bash
pytest
```

## Project Structure

```
modelClassroom/
├── src/modelclassroom/     # Source code
│   ├── tokenizers/         # Tokenizer module
│   ├── models/             # Model operations
│   └── utils/              # Utilities
├── tests/                  # Test suite
├── examples/               # Usage examples
└── docs/                   # Documentation
```
